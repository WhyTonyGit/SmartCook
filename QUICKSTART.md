# Быстрый старт

## Запуск с Docker (рекомендуется)

1. Убедитесь, что Docker и Docker Compose установлены

2. Запустите приложение:
```bash
docker-compose up --build
```

3. В другом терминале загрузите данные:
```bash
docker-compose exec backend python seed.py
```

4. Откройте frontend:
   - Запустите простой HTTP сервер в директории `frontend/`:
   ```bash
   cd frontend
   python -m http.server 8080
   ```
   - Откройте http://localhost:8080 в браузере

## Основные функции

1. **Регистрация/Вход** - создайте аккаунт или войдите
2. **Поиск рецептов** - введите ингредиенты на главной странице или в "Найти рецепт"
3. **Просмотр рецепта** - кликните на карточку рецепта
4. **Избранное** - добавляйте рецепты в избранное
5. **Профиль** - настройте запрещённые ингредиенты
6. **Комментарии и оценки** - оставляйте отзывы о рецептах

## API

Backend доступен на http://localhost:5000

Пример запроса:
```bash
curl http://localhost:5000/api/recipes?ingredients=курица,картофель
```
